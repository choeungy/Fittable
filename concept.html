CONCEPT

Fittable is a smart table top that aims to improve posture through the use of LED lights and an advanced feedback system. The primary objective of Fittable is to encourage users to consistently participate in postural shifts through active breaks and to gain an overall better awareness of posture. The focal point of the project is not on strict body alignment, but rather encouraging movement as a form of active break.

The table top can be fitted onto any existing surface and set up however preferred. A wireless token system is used to facilitate the breaks. The tokens should be placed at different areas of the space at varying distances from the user. These tokens will light up one after another at different time intervals until the final token, accompanied with a vibrating notification, signifies to the user that they should engage in a passive break.

An activity meter on the table top measures how active the user is. When inactive, the meter will turn red and when active the meter will turn green. The LCD screen displays digital cues as further prompt to actively engage with the tokens.

TECHNOLOGIES

The ESP32 is the main component behind the token system with wifi and bluetooth features

This PIR motion sensor detects movement from the upper body and is responsible for the activity meter functionalities

The Arduino is responsible for powering the activity meter and ensuring its functionality

LEDs and NeoPixel rings were used to light up the activity meter and tokens for an ambient effect

The vibration motor was used to send a vibrating notification to the user signalling that they should take a passive break
